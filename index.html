Optical Tomography

Python

A rough attempt at optical tomography. Essentially, images acquired via light transmitted through a transparent or translucent medium are reconstructed into a 3D volume. Using an iPhone camera and a lamp, I was able to reconstruct 3D volumes of a layered phantom and recombine colour channel reconstructions into a single RGB volume.

Logic Circuit Simulator

Python

Stemming from my coursework in data structures, algorithms, and digital logic, I created a basic logic circuit simulator. The backend of the simulator uses a graph-like structure to keep track of logic gates and their connections. The circuit is simulated by starting from the outputs and recursing backwards to the inputs. On the user-facing side, a graphical interface operates via drag-and-drop placement of logic gates and wires. Output is provided in the form of a truth table for the circuit.

Motion Analysis

Python

Developed a toolset for the tracking and kinematics analysis of objects in videos. The tracker allows for a user to define a coordinate system and scale via a graphical interface with drag-and-drop commands, and subsequently position, velocity, and acceleration can be computed and displayed graphically in true spatial coordinates.
