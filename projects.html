MotionAnalysis

Python, OpenCV, MatPlotLib, Git

Developed a toolset for the tracking and kinematics analysis of objects in videos. The tracker allows for a user to define a coordinate system and scale via a graphical interface with drag-and-drop commands, and subsequently position, velocity, and acceleration can be computed and displayed graphically in true spatial coordinates.

Golf Tee Robot

C, EV3 Robot

For my MTE 100 final project, I worked with three classmates to design and program a golf tee resetting robot. The robot quickly and accurately places golf balls on a fixed tee for use at driving ranges. I was primarily responsible for the software component of the project, working to implement key functions and control flow for the main program. I was also responsible for predicting possible errors and adding error checks into the software, along with a running log of errors which had occurred.

YRDSBMarkTools

Python, Mechanize, Git

For this project, I developed a Python library for York Region District School Board students to interact with their grade information from the YRDSB TeachAssist and CareerCruising sites. This was achieved using automated login and scraping using the Mechanize library. Upon retrieval, the grade information could also be output to an Excel file using the XlsxWriter library.
